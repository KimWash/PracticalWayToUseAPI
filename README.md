그래서 사이드프로젝트 하다가 머리 터질거같아서 코딩을 직접 해보려고 합니다.

누군가는 이딴짓 왜햐나고 하실 수 있지만.. 조금이라도 더 효율적인 구조를 찾기 위한 여정이라고 생각해주세요. 저는 근본없이 머리에 피나면서 배우고 있거든요.🤕
